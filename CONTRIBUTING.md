# Contribuer à Bakchich

Bakchich est un projet **source-available** (voir [`LICENSE`](LICENSE)) : le code est
publié publiquement pour la **transparence et l'audit**, mais ce n'est pas un projet
open-source.

## Contributions externes

Les contributions externes (pull requests, forks pour usage) **ne sont pas acceptées**.
Tu peux librement **lire, auditer et analyser** le code.

## Signaler un problème

Les rapports de bug et suggestions sont les bienvenus via les
[issues GitHub](https://github.com/cyril12V/bakchich/issues). Utilise les modèles fournis
et inclus, pour un bug, la sortie de la commande **« Bakchich : Diagnostic »**.

## Sécurité

Pour toute faille de sécurité, **ne pas ouvrir d'issue publique**. Écris à
`privacy@bakchich.dev` avec les détails et les étapes de reproduction.

## Licence commerciale

Pour un usage commercial ou une licence, contacte `privacy@bakchich.dev`.
